# PilahIn - Sistem Manajemen Pengelolaan Sampah

Sebuah sistem informasi manajemen (SIM) yang membantu pengelolaan sampah pada rumah tangga.

## Deskripsi Project

**PilahIn** adalah SIM Pengelola Sampah Rumah Tangga yang dirancang untuk mempermudah proses manajemen pengelolaan sampah, mulai dari pemilahan, pencatatan, hingga monitoring pengelolaan sampah rumah tangga secara lebih terstruktur dan efisien.
